# 기초수학

## 부정적분

계산
1. $\int{x^{n}dx} = \frac{1}{n + 1}x^{n+1} + C$
2. $\int{dx} = x + C$

$\int{f(x)dx} = F(x) + C$
- F(x): f(x)의 한 부정적분
- C: 적분상수

$F^{\prime}(x) = f(x)$
1. $\int{cf(x)dx} = c\int{f(x)dx}$
2. $\int{\{f(x) + g(x)\}}dx = \int{f(x)dx} + \int{g(x)dx}$

## 정적분
$\begin{matrix}
\int_{a}^{b}f(x)dx &=& [F(x)]_{a}^{b} \\
                   &=& F(b) - F(a)
\end{matrix}$
- a: 아래 끝
- b: 위 끝
